Frequently Asked Questions

What Linux distributions work?

Hydroxide supports modern x86_64 distributions, including Arch Linux, Arch-based setups, Debian, and Fedora running stable Wine or Proton configurations.

Is Hydroxide completely safe?

Yes, there are no viruses or malicious elements hidden in the client code. However, altering client-side process data always comes with an inherent risk of basic account flags by game security teams.

How do I launch it?

Once the setup package is downloaded, unpack the binaries into your prefix folder and run the execution script directly from your terminal or wrapper.
